# Panduan Administrator - Website Culturstone

Dokumen ini ditujukan untuk administrator yang bertanggung jawab mengelola konten website Culturstone. Panduan ini mencakup cara masuk ke panel admin dan mengelola berbagai bagian konten.

## 1. Akses dan Login ke Panel Admin

Panel admin adalah pusat kendali untuk semua konten di website.

1.  **Buka Halaman Login:** Buka browser (disarankan Google Chrome atau Firefox versi terbaru) dan akses alamat URL login khusus untuk admin. Biasanya ini adalah `[URL_SITUS_ANDA]/login`.
2.  **Masukkan Kredensial:** Anda akan melihat halaman login. Masukkan **Username** dan **Password** yang telah diberikan kepada Anda.
3.  **Masuk:** Klik tombol **"Login"** atau **"Masuk"**. Jika kredensial benar, Anda akan diarahkan ke dasbor utama panel admin.

## 2. Dasbor Admin

Setelah login, Anda akan disambut oleh dasbor admin. Dasbor ini biasanya berisi:
-   **Sidebar (Menu Samping):** Di sisi kiri, terdapat menu untuk navigasi ke berbagai modul manajemen konten (misalnya: Testimoni, Produk, Portofolio).
-   **Area Konten Utama:** Menampilkan data atau formulir berdasarkan menu yang Anda pilih di sidebar.
-   **Dasbor Pengunjung (`Visitor Dashboard`):** Kemungkinan menampilkan ringkasan atau statistik jumlah pengunjung website.

## 3. Mengelola Konten Website

Berikut adalah cara mengelola setiap jenis konten melalui panel admin.

### 3.1. Mengelola Testimoni

Modul ini digunakan untuk mengatur testimoni yang ditampilkan di halaman publik.

-   **Cara Menambah Testimoni Baru:**
    1.  Di sidebar, klik menu **"Testimonial Management"** atau **"Testimoni"**.
    2.  Cari dan klik tombol **"Tambah Baru"** atau ikon `+`.
    3.  Isi formulir yang muncul: Nama Klien, Isi Testimoni, dan lainnya.
    4.  Unggah foto atau video jika tersedia. Pilih file gambar/video dengan kualitas baik.
    5.  Klik **"Simpan"** atau **"Submit"**. Testimoni baru akan langsung muncul di website.

-   **Cara Mengubah Testimoni:**
    1.  Di halaman manajemen testimoni, cari testimoni yang ingin diubah dalam daftar.
    2.  Klik tombol **"Edit"** atau ikon pensil di sampingnya.
    3.  Ubah informasi pada formulir sesuai kebutuhan.
    4.  Klik **"Simpan"** untuk menyimpan perubahan.

-   **Cara Menghapus Testimoni:**
    1.  Cari testimoni yang ingin dihapus.
    2.  Klik tombol **"Hapus"** atau ikon tempat sampah.
    3.  Akan muncul konfirmasi. Klik **"Ya, Hapus"** untuk mengonfirmasi. Testimoni akan dihapus secara permanen.

### 3.2. Mengelola Produk & Portofolio (Prosedur Umum)

Meskipun modul dapat bervariasi, prosedur untuk mengelola Produk dan Portofolio umumnya sama dengan mengelola Testimoni.

1.  **Pilih Modul:** Klik menu **"Produk"** atau **"Portofolio"** pada sidebar.
2.  **Tambah Data Baru:** Gunakan tombol **"Tambah Baru"** untuk menambahkan item baru, lengkapi formulir (Nama Produk/Proyek, Deskripsi), dan unggah gambar yang relevan.
3.  **Edit/Hapus Data:** Gunakan tombol **"Edit"** atau **"Hapus"** pada setiap item dalam daftar untuk memperbarui atau menghapusnya.

**Penting:** Saat mengunggah gambar, pastikan gambar memiliki kualitas yang baik dan relevan dengan konten untuk menjaga tampilan profesional website.

## 4. Logout

Penting untuk selalu keluar dari panel admin setelah selesai bekerja untuk menjaga keamanan.
1.  Cari tombol **"Logout"** atau **"Keluar"**. Biasanya berada di pojok kanan atas atau di bagian bawah sidebar.
2.  Klik tombol tersebut. Anda akan dikembalikan ke halaman login.
